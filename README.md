# Collection Manager

## Project Overview
A local web-based Collection Manager application that allows users to Create, Read, Update, and Delete records.

Domain: Movie Collection

## Local Run
1. Install XAMPP
2. Start Apache
3. Place this folder inside:
   xampp/htdocs/collection-manager
4. Open a browser and go to:
   http://localhost/collection-manager/

## Features
- Full CRUD functionality
- Data persistence using localStorage
- Starts with 30 records
- Dynamic statistics view
- Delete confirmation
- Input validation

## Technology Used
- HTML
- CSS
- JavaScript
- XAMPP (Apache)
